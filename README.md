![preview](https://ooo.0o0.ooo/2016/03/16/56e9739f6a18a.gif)

## Install

```bash
$ npm install -g imgcat-cli
```

## Usage

```js
$ imgcat http://...

$ imgcat ./path/to/a.gif
```

All [imgcat](https://github.com/egoist/imgcat) options can be used here via CLI flags, like `--width 40px`
